git 克隆GitHub仓库到本地

git clone  HTTPS或SSH链接

--------------------------------------------------------------

git status 查看仓库状态

git add  添加一个文件

git commit 提交到本地

git log  查看提交记录

git log -N   查看N条记录

git log -oneline 查看所有的提交历史记录（每条记录简短显示一行）

git config -l 查看相关的用户名、邮箱等
